# Web Researcher Skill

This skill provides advanced instructions and tools for conducting thorough online research. It goes beyond simple searches to include cross-referencing, source evaluation, and structured summarization.

## Instructions

1.  **Define the Research Goal**: Clearly state the specific questions or topics to be investigated.
2.  **Initial Search**: Use standard search tools to identify key players, concepts, and primary sources.
3.  **Cross-Referencing**: Verify critical information across multiple independent sources.
4.  **Source Evaluation**: Assess the credibility and bias of each source.
5.  **Synthesis**: Combine findings into a coherent, cited report.

## Scripts

- `scripts/search_and_summarize.py`: Automates the initial search and basic summarization process.

## Usage

When tasked with deep research, refer to this skill to ensure a systematic and rigorous approach.
